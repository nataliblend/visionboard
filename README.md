🌟 VisionBoard — Your Future in Focus Quarterly VisionBoard, 100% browser-based. No backend, no account. Data is stored in each person's `localStorage` — private and personal.
✦ Features:

Quarterly boards (Q1–Q4) with year-by-year navigation
Complete history — access any past quarter

Editable cards with:

Title, description / statement

Category (Health, Career, Finances…)

Emoji, accent color

Image via URL or local upload

Progress bar 0–100%

Quarterly intent — guiding phrase at the top

Export / Import JSON — for backup or transfer

Auto-save every 60s (if there are unsaved changes)

Editorial design: Cormorant Garamond typography + Outfit
